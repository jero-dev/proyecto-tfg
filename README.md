La documentación de este proyecto está realizada con LaTeX, por lo tanto para generar el archivo PDF necesitaremos instalar TeXLive (texlive-full) en nuestra distribución.

`
$ pdflatex proyecto.tex
`

`
bibtex proyecto
`

`
pdflatex proyecto.tex
`

O directamente

`
make
